# Site & Account Verification Tool (Zoho CRM)

A tool built in **Zoho CRM** that **verifies websites and account data** to ensure accuracy, completeness, and compliance. Uses **Deluge scripting and API integration** to automate validation of account details and site information, keeping CRM data clean and reliable.

---

## Features

- Verifies if a site is accurate and up-to-date  
- Checks account details for completeness in Zoho CRM  
- Prevents duplicate or invalid records  
- Built with **Deluge scripts and Zoho APIs** for automation  
- Simple to run and integrate within CRM workflows
